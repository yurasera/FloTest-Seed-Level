# FloTest

FloTest adalah aplikasi web sederhana untuk mengelola tes dan soal, serta membantu pengguna mengerjakan tes secara online.

Project ini dibuat sebagai media pembelajaran HTML dan CSS menggunakan dummy data tanpa backend maupun database.

## Tech Stack

* HTML
* CSS

## Scope

### Menggunakan

* HTML
* CSS
* Dummy Data

### Tidak Menggunakan

* PHP
* JavaScript
* Backend
* Database

## Features

### Admin

* Kelola pengguna
* Kelola tes
* Kelola soal

### Mentor

* Kelola tes
* Kelola soal

### Menti

* Kerjakan tes
* Lihat hasil
