# Epic 15 — Acceptance Criteria Review

## Deskripsi

Melakukan review final untuk memastikan semua acceptance criteria terpenuhi.

## Feature: HTML Review

### Tasks

* [ ] Menggunakan semantic HTML (header, nav, main, section, footer, form, table)
* [ ] Tidak menggunakan inline CSS
* [ ] Tidak ada halaman kosong
* [ ] Dummy data konsisten di semua halaman
* [ ] Proper heading hierarchy (h1, h2, h3)
* [ ] Gunakan label pada semua form input

## Feature: CSS Review

### Tasks

* [ ] Typography konsisten di semua halaman
* [ ] Button konsisten dan styled dengan baik
* [ ] Form konsisten (input, select, textarea)
* [ ] Table konsisten
* [ ] Card component konsisten
* [ ] Navigation styling konsisten
* [ ] Responsive design berfungsi

## Feature: Navigation Review

### Tasks

* [ ] Semua link navigasi berfungsi
* [ ] Setiap fitur user dapat diakses
* [ ] Tidak ada broken links
* [ ] Back buttons berfungsi dengan baik

## Feature: Data Review

### Tasks

* [ ] Dummy data tampil sesuai kebutuhan
* [ ] User data konsisten (1 Admin, 2 Mentor, 5 Menti)
* [ ] Test data konsisten (3 tests dengan deskripsi)
* [ ] Question data konsisten dengan test
* [ ] Status values konsisten (Draft, Published, Archived)

## Feature: Functionality Review

### Tasks

* [ ] Semua form dapat diisi
* [ ] Semua button dapat diklik
* [ ] Tidak ada console errors
* [ ] Halaman loading dengan cepat
