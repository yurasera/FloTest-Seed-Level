# Epic 2 Review Questions

## Semantic HTML

1. Apa fungsi elemen `<header>`?
2. Apa fungsi elemen `<nav>`?
3. Apa fungsi elemen `<main>`?
4. Mengapa sebaiknya hanya ada satu `<main>` dalam satu halaman?
5. Apa fungsi elemen `<section>`?
6. Apa perbedaan `<section>` dan `<div>`?
7. Kapan sebaiknya menggunakan `<footer>`?

---

## Layout

1. Mengapa semua halaman menggunakan layout yang sama?
2. Apa keuntungan shared layout?
3. Jika ada perubahan pada header, bagaimana cara memastikan seluruh halaman tetap konsisten?
4. Mengapa layout perlu dipisahkan dari konten halaman?

---

## Typography

1. Mengapa font biasanya didefinisikan pada `body`?
2. Apa fungsi properti `line-height`?
3. Mengapa heading dan paragraph perlu memiliki style yang konsisten?

---

## Container

1. Apa tujuan penggunaan `.container`?
2. Mengapa tidak langsung menggunakan `width: 100%` untuk seluruh konten?
3. Apa fungsi `margin: 0 auto`?

---

## Button

1. Mengapa menggunakan class `.btn`?
2. Apa keuntungan reusable button?
3. Mengapa tombol perlu memiliki padding?
4. Apa fungsi state hover pada tombol?

---

## Form

1. Mengapa setiap input harus memiliki label?
2. Mengapa input menggunakan `width: 100%`?
3. Apa perbedaan `input`, `select`, dan `textarea`?
4. Mengapa form membutuhkan spacing antar field?

## Table

1. Kapan data lebih cocok ditampilkan menggunakan tabel?
2. Mengapa header tabel (`th`) penting?
3. Apa fungsi `border-collapse: collapse`?
4. Mengapa tabel sering membutuhkan wrapper dengan `overflow-x: auto`?

## Card

1. Apa tujuan penggunaan card?

3. Apa manfaat border atau shadow pada card?

## Navigation

1. Apa fungsi navigation pada aplikasi?

1. Apa yang dimaksud responsive design?
